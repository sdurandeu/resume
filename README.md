# my resume
HTML5/js only page. Using Bootrapt 2.3.2.

#### Includes:
 - Schema.org 'Person' tagging for Google search ([more info](http://schema-creator.org/person.php) - [validator](https://developers.google.com/structured-data/testing-tool/))
 - Age calculation
 - Google Analytics
 - Tooltip
